# CLAUDE.MD

## Project Overview

**wanikani-library** is a tool that transforms a WaniKani account into a GPT-friendly library of Japanese vocabulary.

## Project Purpose

This project aims to extract vocabulary data from WaniKani (a Japanese learning platform) and format it in a way that's easily consumable by GPT models and other AI tools for enhanced Japanese language learning and practice.
